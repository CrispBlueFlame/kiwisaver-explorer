# Data dictionary

Tidy CSV copies of the dataset live in [`data/csv/`](data/csv/). They contain the
same values the site uses, in a format ready for Excel, pandas, R, or upload to an
open-data portal. The site itself reads the JSON files in `data/`; the CSVs are a
convenience mirror regenerated by `python3 scripts/build-data.py`.

All fees and returns are percentages (for example `0.85` means 0.85%). All returns
are after fees and are provider-reported to the FMA, not independently audited.
Empty cells mean the value was not reported. Licence: CC BY 4.0. Educational use,
not financial advice.

## funds.csv

One row per current KiwiSaver fund (418 rows). This is the spine of the dataset.

| column | type | meaning |
|--------|------|---------|
| `id` | integer | Stable row id used to join within this dataset. |
| `name` | text | Fund name. |
| `provider` | text | Scheme / provider name. |
| `type` | text | Risk category: Defensive, Conservative, Balanced, Growth, or Aggressive. |
| `fee` | percent | Stated total annual fund fee. |
| `return_1yr` | percent | Most recent 1-year return, after fees. |
| `return_5yr` | percent | Most recent 5-year average annual return, after fees. Blank for funds younger than 5 years. |
| `risk_band` | text | Standard risk indicator range for the fund's type (for example `3-4`). |
| `risk_lo`, `risk_hi` | integer | Numeric ends of that band, on the standard 1 to 7 scale. |
| `horizon_lo`, `horizon_hi` | integer | Suggested investment horizon in years for the fund's type. |
| `ethical` | boolean | True if the fund's name or provider signals an ethical / responsible mandate (keyword-flagged; see caveats). |
| `has_history` | boolean | True if the fund has a matched FMA quarterly series in `fma-history.csv`. |
| `history_since` | year | First calendar year of that series (blank if no history). |

## providers.csv

One row per provider / scheme (36 rows), aggregated from `funds.csv`.

| column | type | meaning |
|--------|------|---------|
| `provider` | text | Scheme / provider name. |
| `funds` | integer | Number of current funds in the dataset for this provider. |
| `avg_fee` | percent | Mean stated annual fee across the provider's funds. |
| `avg_return_5yr` | percent | Mean 5-year return across the provider's funds that report one. |
| `types` | text | Risk categories the provider offers, separated by `;`. |
| `ethical` | boolean | True if any of the provider's funds is ethical-flagged. |

## fma-history.csv

Long format, one row per fund per quarter (3,731 rows) for the 249 funds with a
matched FMA history. Source: FMA Quarterly Fund Updates, September 2013 to
December 2022.

| column | type | meaning |
|--------|------|---------|
| `fund_name`, `provider` | text | Identify the fund; join back to `funds.csv` on name plus provider. |
| `quarter` | date | Quarter-end date, `YYYY-MM-DD`. |
| `fee` | percent | Total annual fund fee reported that quarter. |
| `return_1yr` | percent | 1-year return to that quarter, after fees. |
| `return_5yr` | percent | 5-year average annual return to that quarter, after fees. |
| `risk` | integer | Reported risk indicator (1 to 7) where present. |
| `fum` | NZD | Funds under management for the fund that quarter. |
| `members` | integer | Member count for the fund that quarter. |
| `alloc_cash` | percent | Cash and cash equivalents. |
| `alloc_nz_fixed` | percent | New Zealand fixed interest. |
| `alloc_intl_fixed` | percent | International fixed interest. |
| `alloc_aus_equities` | percent | Australasian equities. |
| `alloc_intl_equities` | percent | International equities. |
| `alloc_listed_property` | percent | Listed property. |
| `alloc_unlisted_property` | percent | Unlisted property. |
| `alloc_other` | percent | Other assets. |
| `alloc_commodities` | percent | Commodities. |

Allocation columns are the percentage of the fund in each asset class and are
blank for quarters where the fund did not report a breakdown.

## early-returns.csv

Pre-2013 market context (65 rows). Morningstar KiwiSaver Survey peer-group
category averages, recovered from Wayback Machine archives. These are category
averages, not individual funds, and cover 2010 to 2013 only.

| column | type | meaning |
|--------|------|---------|
| `category` | text | Morningstar peer group: Conservative, Moderate, Balanced, Growth, or Aggressive. |
| `quarter` | date | Quarter-end date, `YYYY-MM-DD`. |
| `return_1yr` | percent | Category average 1-year return to that quarter. |

## Caveats

- Fees and returns are self-reported by providers to the FMA and are not
  independently audited here.
- `ethical` is a keyword flag on fund and provider names, not an assessment of any
  fund's actual holdings or mandate.
- `risk_band` and the horizon columns are derived from each fund's standard type,
  not from a per-fund figure; per-fund numeric risk indicators in the source could
  not be joined reliably and are omitted from `funds.csv` (they are present in
  `fma-history.csv` where reported).
- No consolidated quarterly fund data exists after December 2022, when the FMA
  stopped publishing it. See `kiwisaver-dataset/DATA_SOURCES.md` for full
  provenance.
